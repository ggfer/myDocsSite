    8. Task Lists

    To create a task list, preface list items with a regular space character followed by []. To mark a task as complete, use [x]. 

    - [x] Finish my changes
- [ ] Push my commits to GitHub
- [ ] Open a pull request
    - [x] Finish my changes - [ ] Push my commits to GitHub - [ ] Open a pull request

    9. Strikethrough

    - [x] Finish my changes
- [ ] Push my commits to GitHub
- [ ] ~~Open a pull request~~
    - [x] Finish my changes - [ ] Push my commits to GitHub - [ ] Open a pull request

    10. Tables

    To create a table, use three or more hyphens --- to create each column’s header, and use pipes | to separate each column. You can optionally add pipes on either end of the table.

    | Name    | Age | Blog |
| ------- | --- | ---- |
| Peter   | 20  |[Peter](http://www.baidu.com) |
| Trump   | 33  |[Trump](http://www.google.com)|
    Name Age Blog
    Peter 20 Peter
    Trump 33 Trump

    11. Images

    To create an image, use ![image-alt](image-href 'image-title')
    The alt attribute provides alternative information for an image if a user for some reason cannot view it.
    The title attribute is shown along with the image, typically as a hover tooltip. 

    ![picture-name](screenshots/architecture.png 'architecture-title')
